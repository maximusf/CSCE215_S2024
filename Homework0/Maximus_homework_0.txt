Research
1. grep
Searches a file for a pattern of characters and displays all lines that contain the pattern

2. find
Allows users to search by name, size, modificaiton time, or content in file exploration.

3. head 
Prints the top N number of data of the given input. It defaults to printing the first 10 lines of the specified files.

4. tail
Prints the last N number of data of the given input. It defaults to printing the last 10 lines of the specified files.

5. sed
Stands for stream editor and can perform functions on a file like searching, find/replace, insertion/deletion. 

6. "top" or "htop"
"top" - Used to show Linux processes by providing a dynamic real-time view of the running system.
"htop" - Newer program compared to top. Used to monitor system's vital resources or server's processes in real time.

7. "vi or "vim"
An advanced and highly configurable text editor built to enable efficient text editing.

8. rm
command is used to remove files, directories, symbolic links, and more within the file system.

9. fortune
Prints out a random interesting proverb.

10. yes
Used to print a continuous output stream of a given STRING.
